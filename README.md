# SWOT Uganda Tool
An interactive web application that displays SWOT river data and discharge estimations for the Upper Victoria Nile river in Uganda.

## Setup Required
This application cannot be run by simply opening a HTML file. Follow the instructions to load data files and access the web application.

## INSTRUCTIONS
1. Download or clone this repository
2. Open Terminal/Command Prompt
3. Navigate to the project folder: `cd path/project`
4. Install the extensions: `python -m pip install flask pandas geopandas`
5. Run the python code: `python app.py`
6. Open your browser and go to: [http://127.0.0.1:5000/]

## About
This web app was developed as part of a Master thesis for the Geo-information Science and Earth Observation programme at ITC, University of Twente (2026).
